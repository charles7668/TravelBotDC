# ✈️ TravelBotDC 旅行助手

> [!WARNING]
> ⚠️ **開發中 (Development in Progress)**：本專案目前處於積極開發階段，功能與指令可能會隨時變動。

這是一個專為旅行團隊設計的 Discord 機器人，支援精確的旅程規劃、多人協作標記、詳細筆記編輯與自動化天氣觀測功能。

---

## 🛠️ 主要功能指令

本機器人全面使用 Discord 斜線指令 (`/`)，輸入時會提供智慧參數建議。

### 🗓️ 1. 旅程規劃 - Planner (團隊協作)

*   **/create_trip** `name:` `start_date:` `end_date:`
    *   **用途**：建立一個新的大型旅程主項目。
*   **/join_trip** `name:`
    *   **用途**：加入現有的旅程計畫。加入後，該旅程的所有提醒都會自動標記 (@) 您。
*   **/schedule** `time_str:` `task:` `trip_name:` (選填) `location:` (選填) `description:` (選填)
    *   **用途**：排定細部行程。支援 `YYYY-MM-DD` 或 `YYYY-MM-DD HH:MM` 格式。
    *   **智慧聯動**：若行程在今日，Bot 會即時回報當地的天氣狀況。
*   **/view_trip** `name:`
    *   **用途**：檢視特定旅程。回應包含**「互動式按鈕」**，點擊即可查看各項行程的詳細描述或地圖連結。
*   **/list_trips**
    *   **用途**：列出目前參與的所有旅程概覽（僅個人可見）。

### 📝 2. 詳細筆記與管理 - Schedule Editor

*   **/view_details** `full_task_id:`
    *   **用途**：查看特定行程的詳細備註。支援**「Markdown 排版」**（如：可點擊的超連結、粗體、清單）。
*   **/edit_schedule** `full_task_id:`
    *   **用途**：修改行程內容。更新後會在頻道中發布**「詳細對比公告」**（顯示舊內容 ➔ 新內容）。
    *   **進階編輯**：若需輸入多行長文本，會彈出**「詳細專用視窗」**進行編輯。
*   **/delete_schedule** `full_task_id:`
    *   **用途**：移除行程。具備**「防誤刪按鈕」**，需透過二次點擊確認後方可刪除。

### ⏰ 3. 工具功能 - Weather & Timers

*   **/weather** `city:`
    *   **用途**：即時查詢全球城市的天氣預報（包含溫度、風向與降雨描述）。
*   **/remind** `reminder_time:` `task:`
    *   **用途**：設定當天內的定時鬧鐘提醒。

---

## 🔔 智慧通知機制

*   **團隊導覽 (Group Mentions)**：凡是加入旅程 (`/join_trip`) 的成員，都會在該旅程相關的倒數通知中被自動標記。
*   **提前倒數 (Countdown)**：針對行程項目，系統會於 **提前 3 天** 與 **提前 1 天** 的 **凌晨 00:00** 自動發送預告。
*   **當日報時 (Weather Report)**：行程當天凌晨，Bot 會主動檢測該項目的地點天氣，並彙整所有成員及其行程發送一份「天氣 + 行程」日報。

## 🚀 快速開始

1.  **安裝依賴**：
    ```bash
    uv sync
    ```
2.  **設定 token**：
    在 `.env` 中填入您的 `DISCORD_TOKEN`。
3.  **啟動機器人**：
    ```bash
    uv run bot.py
    ```
4.  **強制刷新指令 (開發者專用)**：
    若指令未出現，請在頻道輸入 `!sync` 並重啟您的 Discord 客戶端。
