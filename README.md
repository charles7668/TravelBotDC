# ✈️ TravelBotDC 旅行助手

> [!WARNING]
> ⚠️ **開發中 (Development in Progress)**：本專案目前處於積極開發階段，功能與指令可能會隨時變動。

這是一個專為旅行者設計的 Discord 機器人，提供即時天氣查詢、定時任務提醒與長途行程倒數功能。

---

## 🛠️ 開始使用

### 1. 安裝環境與啟動
1. 確保已安裝 [uv](https://docs.astral.sh/uv/)。
2. 開啟終端機並同步環境：
   ```bash
   uv sync
   ```
3. 啟動機器人：
   ```bash
   uv run bot.py
   ```

### 2. 設定 Token (金鑰)
建立一個 `.env` 檔案並填入您的金鑰：
```text
DISCORD_TOKEN=在此貼上您的_BOT_TOKEN
```

---

## 📖 指令列表 (Slash Commands)

機器人啟動後，直接在 Discord 輸入欄中鍵入 `/` 即可看到以下指令：

### 🌤️ 天氣查詢
*   **/weather** `city:`
    *   **用途**：查詢全球城市目前的氣溫與天氣狀態。
    *   **支援中文**：例如 `/weather 台北`、`/weather 東京`、`/weather 倫敦`。

### ⏰ 今日定時提醒
*   **/remind** `reminder_time:` `task:`
    *   **用途**：設定當天內部的具體時間提醒。
    *   **格式**：HH:MM。
    *   **範例**：`/remind 14:30 前往機場登機`。

### 📅 旅行行程排定 (長期倒數)
*   **/schedule** `date:` `task:`
    *   **用途**：排定未來的長期行程，Bot 會自動在關鍵時間點發送提醒。
    *   **格式**：YYYY-MM-DD。
    *   **範例**：`/schedule 2024-12-31 到紐約跨年`。
    *   **自動提醒機制**：
        1. **提前 3 天**：Bot 會主動發送通知，提醒您準備行李或檢查文件。
        2. **提前 1 天**：Bot 會強力提醒您明天就要出發。
        3. **出發當日**：祝您旅途愉快！

## 💡 注意事項
