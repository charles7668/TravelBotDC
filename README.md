# ✈️ TravelBotDC 旅行助手

> [!WARNING]
> ⚠️ **開發中 (Development in Progress)**：本專案目前處於積極開發階段，功能與指令可能會隨時變動。

這是一個專為旅行團隊設計的 Discord 機器人，支援精確的旅程規劃、多人協作標記與自動化天氣觀測功能。

---

## 🛠️ 主要功能指令

本機器人全面使用 Discord 斜線指令 (`/`)，輸入時會提供智慧參數建議（Autocomplete）。

### 🗓️ 1. 旅程規劃 - Planner (長期與團隊)
*   **/create_trip** `name:` `start_date:` `end_date:`
    *   **用途**：建立一個新的大型旅程主項目。
*   **/join_trip** `name:`
    *   **用途**：**[新功能]** 加入現有的旅程計畫，之後該旅程的所有提醒都會自動標記 (@) 您。
*   **/schedule** `time_str:` `task:` `trip_name:` (選填) `location:` (選填)
    *   **用途**：排定細部行程。支援 `YYYY-MM-DD` 或 `YYYY-MM-DD HH:MM` 格式。
    *   **智慧聯動**：若填寫 `location`，Bot 會在行程當日自動回報該地天氣。
*   **/view_trip** `name:` (選填)
    *   **用途**：檢視特定旅程中的行程、參與成員清單與每日計畫。
*   **/list_trips**
    *   **用途**：列出目前所有的旅程、參與人數與分組概覽。

### ⏰ 2. 快速提醒 - Timers (今日)
*   **/remind** `reminder_time:` `task:`
    *   **用途**：設定當天內的定時鬧鐘（`HH:MM` 格式）。

### 🌤️ 3. 工具功能 - Weather
*   **/weather** `city:`
    *   **用途**：即時查詢全球城市的天氣預報（包含溫度、風速與降雨狀況描述）。

---

## 🔔 智慧通知機制

*   **團隊導覽 (Group Mentions)**：凡是加入旅程 (`/join_trip`) 的成員，都會在該旅程相關的倒數通知中被自動標記。
*   **提前倒數 (Countdown)**：針對行程項目，系統會於 **提前 3 天** 與 **提前 1 天** 的 **凌晨 00:00** 自動發送預告。
*   **當日報時 (Weather Report)**：行程當天凌晨，Bot 會主動檢測該項目的地點天氣，並彙整所有成員及其行程發送一份「天氣 + 行程」日報。

---

## 🚀 快速開始

1.  **安裝依賴**：
    ```bash
    uv sync
    ```
2.  **設定 token**：
    在 `.env` 中填入您的 `DISCORD_TOKEN`。
3.  **啟動機器人**：
    ```bash
    uv run bot.py
    ```
4.  **強制刷新指令 (開發者專用)**：
    若指令未出現，請在頻道輸入 `!sync` 並重啟您的 Discord 客戶端。
