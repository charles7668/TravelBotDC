# ✈️ TravelBotDC 旅行助手

> [!WARNING]
> ⚠️ **開發中 (Development in Progress)**：本專案目前處於積極開發階段，功能與指令可能會隨時變動。

這是一個專為旅行者設計的 Discord 機器人，提供精確的旅程規劃、智慧提醒與全方位的天氣預報功能。

---

## 🛠️ 主要功能指令

本機器人全面使用 Discord 斜線指令 (`/`)，輸入時會提供智慧參數建議（Autocomplete）。

### 🗓️ 1. 旅程規劃 - Planner (長期)
*   **/create_trip** `name:` `start_date:` `end_date:`
    *   **用途**：建立一個大型旅程區間（例如：東京賞櫻行）。
*   **/schedule** `time_str:` `task:` `trip_name:` (選填)
    *   **用途**：在特定日期或時間排定各類小行程。
    *   **時間格式**：支援 `YYYY-MM-DD` (全天) 或 `YYYY-MM-DD HH:MM` (精確點)。
    *   **智慧歸類**：若不填 `trip_name`，會自動歸類至「未分組」。
*   **/view_trip** `name:` (選填)
    *   **用途**：檢視特定旅程中的詳細行程清單。預設顯示「未分組」內容。
*   **/list_trips**
    *   **用途**：列出目前所有的旅程與分組概錄。

### ⏰ 2. 快速提醒 - Timers (今日)
*   **/remind** `reminder_time:` `task:`
    *   **用途**：設定當天內的定時鬧鐘。
    *   **時間格式**：`HH:MM` (例如：`14:30`)。

### 🌤️ 3. 工具功能 - Weather
*   **/weather** `city:`
    *   **用途**：查詢全球城市的天氣預報。
    *   **特色**：完美支援中文地名輸入（如：台北、東京）。

---

## 🔔 自動通知機制

*   **長期倒數**：針對您在 `/schedule` 中排定的項目，系統會於 **提前 3 天** 與 **提前 1 天** 的 **凌晨 00:00** 自動發送預告，確保您做好準備。
*   **定時提醒**：`/remind` 指令設定成功的鬧鐘，會在時間準確到達的小時/分鐘標註使用者。

---

## 🚀 快速開始

1.  **安裝依賴**：
    ```bash
    uv sync
    ```
2.  **設定環境變數**：
    在 `.env` 中填入您的 `DISCORD_TOKEN`。
3.  **啟動機器人**：
    ```bash
    uv run bot.py
    ```
4.  **強制刷新指令 (開發者專用)**：
    若指令未出現，請在頻道輸入 `!sync` 並重啟您的 Discord 客戶端。
